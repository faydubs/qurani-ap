## Packages
framer-motion | Complex animations for the crescent moon, stars, and progress indicators
clsx | Conditional class merging for dynamic styles
tailwind-merge | Utility for merging Tailwind classes safely

## Notes
The application requires a dark, spiritual theme with support for RTL text layout.
Fonts will be loaded via Google Fonts in index.css (Amiri/Cairo).
Authentication endpoints are standard session-based.
